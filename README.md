Nombre del proyecto: PlanAR   
**Integrantes:** Coccia Agustín, Soria Esteban, Gaitan Morena.     

**Descripción:** Aplicación para que las personas puedan descubrir eventos, recitales, ferias y actividades culturales.          

**Tecnologías utilizadas:** CSS, HTML, JavaScript, Git, GitHub, QA e Infraestructura.     

**Funcionalidades desarrolladas:** 
eventos.html: Título principal.
              Breve descripción.
              Tres eventos disponibles.
              Botón "Reservar lugar".
              Botón "Mostrar categorías".
              Botón "Volver al inicio".
              Nombre del evento.
              Fecha.
              Lugar.
              Flexbox.
              Colores.
              Border-radius.
              Box-shadow.
              Efecto hover sobre las tarjetas.

Error encontrado: El botón "Reservar lugar" se puede presionar múltiples veces.
Pasos para reproducirlo: 1. Abrir la página. 2. Hacer clic en "Reservar lugar". 3. Volver a hacer clic.
Resultado esperado: El botón debería deshabilitarse tras la primera reserva.
Solución aplicada: Deshabilitar el botón después del primer clic.
